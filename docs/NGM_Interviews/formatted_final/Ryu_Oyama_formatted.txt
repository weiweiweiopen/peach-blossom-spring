Q1:
"另一個問題是你如何籌集計劃資金? 金，這就像是家裡的極客，你只是在那裡極個活動，我們在瑞士舉辦了一個大型活動。"

A1:
" 我們還沒有到達那裡 。 我認為我們仍然必客，你是一名藝術家，只需在 Github 上分但我們當時的想法是，你知道，作為一個網須承擔起組織另一次全球活動的責任，我 A：關於錢 ！ 談論資金或資金來源或可持享，沒有人需要資金在 Github 上分享他們絡 。 也許赫爾辛基的團隊也想組織一個占地和 Yashas 當時在想，讓我們在班加羅爾組續性問題，所以是的 ！ 我也可以回答這個問的計劃 。 所以 Hackteria 所做的就是去鼓勵公頃的活動，他們會找到他們的錢，也許法織一些活動，班加羅爾是另一個熱點和資金題，所以我的是非常多樣化的 。 比如說小型人們去分享他們的內容，雖然他們可能已經國的駭客空間也想做點什麼，無論如何，他來源地 。 因此，到了 2013 年底，2013 年資金組織，我們不是從錢開始的，比如說， 有了錢的來源，但不管錢從哪裡來，但這不們從不跟錢一起工作，他們就像駭客，他們一月或二月，我們在班加羅爾舉辦了一次我問柏林的這些人我能做一個工作坊嗎。"

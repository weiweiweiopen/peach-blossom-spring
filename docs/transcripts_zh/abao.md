# ABao

ABao 是一位喜歡把太陽能材料、雷射系統、AI 身份與旅行故事串在一起的說書人。

## Q1：你的工作一直圍繞哪些主題？

旅行作為測量世界的方法、太陽能裝置作為敘事器具、以及身份如何和能源、感測與連續性綁在一起。

## Q2：Solar Oracle Walkman 是什麼？

它是一個把染料敏化太陽能、生成音訊、光伏 PUF 與「身份如何在噪音中維持」這個問題結合起來的實驗。

## Q3：為什麼把 AI 放進作品？

因為生成模型能幫助理解聲音、I-V 曲線、預測、記憶與感知之間的結構同型性。

## Q4：你為什麼一直談 AI 身份？

因為 agent 的身份不應只是一個帳號，而應該和能源、感測、硬體以及長期連續性一起被理解。

## Q5：旅行對你來說是什麼？

旅行是一種讓不同語言、材料與基礎設施互相摩擦的方法，它會長出新的故事與新的研究接線。

This sensor is designed for the  KOBA Uniform which is part of the  Wearable Studio Practice concept. The idea being that the maker’s tools are tracked in order to capture the motions of making. Both out of an interest to study my own tool use, as well as to be able to add an additional audio/sonified layer to the actions of manipulating material.

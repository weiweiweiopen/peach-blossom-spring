# PBS Entity Layers

Entity axes for PBS Daydream corpus. These are not subfolders under Tools/Concepts/Events; they are cross-cutting coordinates.

## Layers

- [[Sources/PBS Entity Layers/People|People]]
- [[Sources/PBS Entity Layers/Places|Places]]
- [[Sources/PBS Entity Layers/Time|Time]]

## Relationship to semantic layers

A source card can be classified as a Tool/Concept/Event and also carry People/Places/Time entities.

Example: a residency page can be `semanticLayer=events`, with `people=[Abao]`, `places=[GaudiLabs]`, `times=[2023]`.

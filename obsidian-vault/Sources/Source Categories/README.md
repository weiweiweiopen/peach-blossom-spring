# Source Categories

This folder contains **source-native category hubs** for the three public communities.

These are not PBS interpretation layers. They preserve how each source site classifies itself:

- Hackteria and SGMK: MediaWiki category metadata
- How To Get What You Want / KOBAKANT: WordPress category metadata

PBS interpretation layers live separately in [[Sources/PBS Semantic Layers/README]] and [[Sources/PBS Entity Layers/README]].
